# XR WVM Fullstack Developer Capstone Project

## Project Overview

This project is a Django + React full-stack web application for managing car dealerships, cars, and dealer reviews.

## Repository

- Backend: Django REST API
- Frontend: React.js
- Database: SQLite (for development)
- Deployment: IBM Code Engine / Render

## Features

1. User Authentication (Login, Logout, Register)
2. Dealer CRUD operations
3. Car and Car Make listings
4. Dealer Reviews and Sentiment Analysis
5. Responsive Frontend with About/Contact Pages
6. CI/CD workflow with GitHub Actions
